## typeOf
undefinedの値にtypeof演算子を用いると **必ず”undefined”** が返る

<br>

(例)`text`が`undefined`だった場合処理を実行
```js
let text;
 
if (typeof text === "undefined") {
  処理;
}
```

<br>

## void
void演算子は与えられた式に対して **常にundefined** を返す
(どのような数値や文字列でも`undefined`になる)
  
(例)`text`が`undefined`だった場合処理を実行
```js
var text;
 
if (text === void 0) {
  処理;
}
```
