## セレクタを指定
  
### HTML タグを指定
(例)`div`タグを指定
```js
$('div')
```

### 階層的に指定
(例)`div`タグ内の`img`タグ
```js
&('div > img')
```

### id で指定
(例)`id="preview"`を持つ要素を指定
```js
$('#preview')
```

### class で指定
(例)`class="preview"`を持つ要素を指定
```js
$('.preview')
```

### 属性で指定
#### HTML要素[属性　＝ 値]
(例)`input`属性を取得
```js
$('input[type=file]')
```
