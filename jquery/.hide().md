# .hide()
指定した要素を非表示にすることができる。

引数には非表示にする際の時間や、関数を設定できる
  
[jQuery 日本語リファレンス](https://js.studio-kingdom.com/jquery/effects/hide#5)

<br>

## 要素を非表示にする
　　
(例)`img`タグを非表示にする
```js
$('img').hide();
```

(例)`id="preview"`を持つ HTML を非表示にする
```js
$('#preview').hide();
```

<br>

### 時間を指定して非表示にする

```js
$('img').hide('slow');
```
