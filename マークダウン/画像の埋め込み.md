# 画像の埋め込み
```
![テキスト](画像の URL)
```
(例)GitHub の画像
  
```
![GitHub の画像](https://cdn.icon-icons.com/icons2/509/PNG/512/Github_icon-icons.com_49946.png)
```
![GitHub の画像](https://cdn.icon-icons.com/icons2/509/PNG/512/Github_icon-icons.com_49946.png)
  
(例) 画像のサイズを指定する(横幅 200px)
  
```
<img src="画像の URL" arl="テキスト" width="横幅">
```
