# Chartkick
  
<br>
  
## インストール
`Gemfile`に追記
```
gem 'chartkick'
```
  
### `chart.js`を使用する(Rails6)
```
yarn add chartkick chart.js
```
`app/javascript/packs/application.js`に追記
```rb
import "chartkick/chart.js"
```

<br>

### `highcharts`を使用する（Rails6）
```
yarn add chartkick highcharts
```
`app/javascript/packs/application.js`に追記
```rb
import "chartkick/highcharts"
```
