# sum
## 要素の合計を求める
```rb
numbers = [1, 2, 3]

numbers.sum
#=> 6
```
  
<br>
  
## ブロックの戻り値の合計を求める
(例)配列の要素を2倍にしながら合計を求める
```rb
numbers = [1, 2, 3]

numbers.sum {|n| n * 2 }
#=> 12
```
  
<br>
  
## 初期値を指定して合計を求める
デフォルトの初期値は`0`
(例)初期値に`10`を渡して合計を求める
```rb
numbers = [1, 2, 3]

numbers.sum(10)
#=> 16
```
  
<br>
  
## 数値ではなく文字列を指定する
初期値に文字列を指定すると各要素の文字列が連結される
(例)文字列が`a + b + c`となり連結される
```rb
chars = ['a', 'b', 'c']

chars.sum('')
#=> 'abc'
```
  
## ブロック内で文字列を加工する
(例)初期値に`>`を指定して、ブロックの戻り値を`大文字`に変換して返す
```rb
chars = ['a', 'b', 'c']

chars.sum('>') {|s| s.upcase }
#=> '>ABC'
```
